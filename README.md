# find-color
FindColor is a browser-based tool that extracts colors from images using K-Means++ clustering, analyzing up to 50,000 pixels with RGB distance matching. It includes color harmony tools, outfit analysis, a CIELAB (ΔE) heatmap, and a persistent color wallet. Built with vanilla JavaScript, it supports 16 languages and dark/light mode.
